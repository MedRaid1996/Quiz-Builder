# Quiz Builder

Application React de création et correction de questionnaires.

## Lancer le projet
```bash
npm install
npm run dev
```

## Fonctionnalités réalisées

- [ ] Création du quiz (titre via formulaire non contrôlé)
- [ ] Ajout de questions (formulaire contrôlé)
- [ ] Validation des formulaires
- [ ] Affichage des questions créées
- [ ] Passage du quiz
- [ ] Correction et score final
- [ ] Bonus : react-hook-form
- [ ] Bonus : localStorage
- [ ] Bonus : minuterie
